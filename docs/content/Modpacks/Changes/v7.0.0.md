# Updating from 1.6.x to 7.0.0

## Machine Casing Models

If you are registering custom machines using KubeJS, you need to replace the call to `.workableCasingRenderer(...)` with `.workableCasingModel(...)`

Please note that the third argument needs to be removed, so the following changes need to be made:

```diff
  GTCEuStartupEvents.registry('gtceu:machine', e => {
      e.create('greenhouse', 'multiblock')
          //...
-         .workableCasingRenderer('gtceu:block/casings/solid/machine_casing_solid_steel', 'gtceu:block/multiblock/implosion_compressor', false)
+         .workableCasingModel('gtceu:block/casings/solid/machine_casing_solid_steel', 'gtceu:block/multiblock/implosion_compressor')
  })
```
