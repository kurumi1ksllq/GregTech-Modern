## 关于本 Mod

本 Mod 是 [GregTech CEu](https://github.com/GregTechCEu/GregTech-Modern) 的 GTNH TNG 特供版本，进行了部分定制化修改。

## 开发说明

### IDE 要求

- [Lombok Plugin](https://plugins.jetbrains.com/plugin/6317-lombok) (IntelliJ IDEA 必需)
- [Minecraft Development Plugin](https://plugins.jetbrains.com/plugin/8327-minecraft-development) (推荐)

### 构建

```bash
./gradlew build
```

### 运行开发客户端

```bash
./gradlew runClient
```
